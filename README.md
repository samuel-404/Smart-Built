Project Overview: SmartBuildSmartBuild is an intelligent, hybrid expert system designed to automate the complex process of custom PC configuration. Developed by a team consisting of Irene Ann Thomas, Naveen D, Samuel B, and Vyshnav S under the guidance of Dr. Ajish S, the project aims to "democratize" high-performance computing by making expert-level hardware recommendations accessible to novice users.
Core Methodology: The Hybrid Inference EngineThe system's "intelligence core" relies on a two-phase process that combines two distinct Al approaches to ensure both hardware compatibility and performance optimization:Phase 1: Constraint Satisfaction (The Gatekeeper)Algorithm: Arc Consistency Algorithm #3 (AC-3).Function: It models PC building as a Constraint Satisfaction Problem (CSP). It manages variables such as the CPU, GPU, Motherboard, PSU, RAM, Storage, and Case.Impact: By enforcing hard constraints—such as socket compatibility (e.g., LGA1700) and TDP versus PSU wattage—it prunes incompatible component combinations early. This reduces the search space from $10^{15}$ to $10^8$ combinations, transforming an intractable problem into a manageable one for the optimizer.Phase 2: Evolutionary Optimization (The Optimizer)Algorithm: Non-dominated Sorting Genetic Algorithm II (NSGA-II).Function: This phase handles "soft objectives," specifically maximizing performance while minimizing the price deviation from the user's budget.Outcome: The algorithm typically converges to a Pareto-optimal front within 40–50 generations. This allows the system to present the top 5 configurations that represent the best trade-offs between cost and performance.

Key Features & AdvantagesGuaranteed Compatibility: Provides mathematical assurance that all selected parts will function together seamlessly.Strict Budget Adherence: Uses penalty functions within the Genetic Algorithm to strictly respect the user's defined budget.Personalized Use Cases: Tailors builds based on specific needs, such as Gaming, Content Creation, Student/General use, or Workstations.High Efficiency: The system achieves an average latency of less than 2 seconds for a full optimization cycle.
